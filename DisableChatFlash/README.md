# Disable Chat Flash

A World of Warcraft addon for **Burning Crusade Classic Anniversary** that disables the flashing notification on chat tabs when receiving whispers.

Designed to keep your chat interface clean and distraction-free.

## Installation

1.  Download the repository.
2.  Copy the `DisableChatFlash` folder to your WoW AddOns directory:
    -   `_classic_\Interface\AddOns\`
3.  Restart WoW or reload your UI (`/reload`).
